01

Fabric Migration Assistant for Data Warehouse

Synapse DW Schema/Tables
40-55% Productivity Gain

What It Is

Native GA tool built into the Fabric portal providing a guided, four-step migration experience for moving Synapse dedicated SQL pools into Fabric Data Warehouse. Microsoft offers personalized migration support through their Migration Factory at no cost.

How It Works

  1. Metadata Migration via DACPAC: Extract schema definitions using Visual Studio, VS Code, or SqlPackage CLI
  2. AI-Assisted Problem Resolution: Copilot analyzes broken scripts and suggests fixes with inline comments
  3. Data Copy via Fabric Data Factory: Integrated copy jobs for one-time full or continuous incremental copying
  4. Connection Rerouting: REST API identifies and reroutes connections to new Fabric warehouse

02

ADF-to-Fabric PowerShell Pipeline Upgrade Module

ADF Pipelines/Datasets
30-45% Productivity Gain

What It Is

Microsoft's Microsoft.FabricPipelineUpgrade PowerShell module automates migration of ADF pipelines, datasets, linked services, and triggers into Fabric Data Factory. Complemented by a built-in Migration Assessment Tool.

How It Works

  1. Assessment: Run upgrade assessment to categorize pipelines as Compatible, Needs Review, or Not Compatible
  2. Import: Import-AdfFactory cmdlet imports pipeline definitions as JSON
  3. Convert: ConvertTo-FabricResources translates to Fabric-native formats
  4. Resolve: JSON resolution file maps linked services to Fabric connections
  5. Export: Export-FabricResources deploys to target workspace

Stored Procedure Conversion

What It Is

The highest-impact Deloitte-built accelerator. Microsoft doesn't provide turnkey T-SQL-to-PySpark conversion, but Fabric notebooks with T-SQL magic commands and the Spark Data Warehouse Connector enable hybrid approaches.

Simple

Spark SQL passthrough - straightforward SELECT, INSERT, UPDATE, DELETE, MERGE

Medium

PySpark DataFrames - dynamic SQL, cursor logic, conditional branching

High

Full refactoring - cross-DB references, complex error handling, Synapse-specific features

05

Automated Data Validation and Testing Framework

Cross-Cutting Quality
Quality Risk Reduction

What It Is

A critical gap filler - the base estimate doesn't include dedicated data validation budget. This framework systematically validates data integrity, completeness, and accuracy between source and target environments.

Validation Levels

1
Schema Validation

Automated comparison of column names, data types, nullability, constraints

2
Row-Level Validation

Automated row counts, checksum/hash validation, sample record comparison

3
Business Rule Validation

Pre-defined test cases for critical calculations, aggregations, derived values

Net Impact

Paradox: This accelerator adds scope (5-8% = 800-1,350 hours) while enabling 10-15% reduction in SIT/UAT rework. Net result is quality improvement and reduced production incidents.

DMTSP Buildout

Build Effort: Medium

  • Parameterized PySpark notebooks for schema comparison
  • Row count and checksum validation with configurable thresholds
  • Sample record comparison with fuzzy matching
  • Business rule test case templates
  • HTML/Power BI validation report generator
